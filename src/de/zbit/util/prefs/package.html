A collection of classes for dealing with user-defined options and configuration key, either from the command line or from some system configuration. Thereby, three layers are considered:
  1. User options from the command line
  2. Options that are currently in memory but not yet persistent
  3. Persistently saved user configuration
The idea is to define a key-value pair for the option and an associated user-defined value and to re-use this pair in graphical user interfaces, on the command line and also at other positions within the program.
